# 静态项目配置

```js
npm init
```

```js
yarn add webpack@4.44.1 webpack-cli@3.3.12 webpack-dev-server@3.11.0
```

```js
yarn add html-webpack-plugin@4.3.0
```



根目录增加配置文件webpack.config.js

```js
const htmlWebpackPlugin = require('html-webpack-plugin')
module.exports = env => {
    return {
        entry: './public/index.js',
        output: {
            filename: 'app.js'
        },
        devServer: {
            host: 'test.m.iqiyi.com',
            port: 8080,
        },
        plugins: [
            new htmlWebpackPlugin({
                template: './src/views/vip/index.html',
                filename: 'index.html',
            }),
        ]
    }
}
```
